## ğŸš€ How to Get Added

### Automatic Recognition

When your PR is merged, our bot automatically adds you to the contributors list! Just make sure your PR is properly labeled.

### Manual Addition

You can also request to be added by commenting on any issue or PR:

```
@all-contributors please add @username for code, doc
```

Replace `@username` with the GitHub handle and specify contribution types (comma-separated).

### Update Your Contributions

Already listed but made new contributions? Comment:

```
@all-contributors please add @username for security
```

See [CONTRIBUTING.md](./CONTRIBUTING.md) for detailed contribution guidelines.
